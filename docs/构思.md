# 整体构思
___
## 界面
### 时间
* 当前时间
* 番茄钟
* 当前天气(API: 心知天气)
* 每日名句

### 原神 Genshin Impact拓展
* 当前树脂数量
* 参量质变仪
* 每日委托

### 工具栏
* 常用软件
* 调整音量 / 音乐
